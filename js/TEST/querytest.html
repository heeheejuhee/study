Logic Tips

  1. Make sure you have everything wrapped in DOMContentLoaded event
  2. Query select buttons (SELECT BUTTONSSSSSS) and the circle
  3. Loop all buttons
  4. For every buttons, add event listener 'click' (buttons[i].addEventListener...)
  5. When you click each button, check it's class name.(Try console.log(this.className))
  6. If the button has class top OR bottom, set style.top and style.bottom to 'auto'
  7. else if the button has class left OR right, set style.left and style.right to 'auto'
  8. If the button has class top, set style.top = 0. repeat else if for bottom/left/right
