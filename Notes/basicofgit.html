Terminal

CLI : (COMMANDLINE INTERFACE)

pwd : where am i?
clear : clear
open . : Opening folder

ls : listing everything
ls -la : listing in detail

cd : change directory

DIRECTORY

mkdir : Make directory
rmdir : Remove directory
rm -rf : Remove directory with file

FILE

touch : Create file
rm : remove file

ctrl+c : cancel

example:

cd Desktop
mkdir JuheeFolder
rmdir JuheeFolder
touch index.html

GIT

git init : Starting Git
git status /gst: Current project status
git log /glo: Logs of all commits
git diff /gd: Look at the change

git add /ga: Adding a file/s to staging
git commit /gc: Commit file/s
git commit -m "Message~"
git push /gp: Push file/s
